  - For a differential equation we have a variational principle.

    \[ E[\Psi] = \expval{\hat{H}}{\Psi} / \expval{\Psi} \]

  - If we vary the value of $\Psi$ we can find the ground state if the
    *functional* $E[\Psi] \geq E_0$.
